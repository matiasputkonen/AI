# My Agents

This folder contains **personal AI agents** developed for experimentation, productivity, and automation.

The agents here are designed to assist with **research, development, decision-making, and daily workflows**. Each agent focuses on a specific task and can evolve into more advanced autonomous systems over time.

---

## Purpose

The goal of this folder is to:

* Develop practical **AI agents for personal use**
* Experiment with **autonomous and tool-using AI systems**
* Build modular agents that can interact with APIs, files, and data
* Explore modern **AI agent architectures**

These agents may support activities such as research, coding, automation, and strategic planning.

---

## Structure

Each agent should be placed in its own folder.

Example structure:

```
My agents
│
├── research_agent/
│   ├── agent.py
│   ├── tools.py
│   └── README.md
│
├── workflow_agent/
│   ├── agent.py
│   ├── tasks.py
│   └── README.md
│
└── knowledge_agent/
    ├── agent.py
    ├── memory.py
    └── README.md
```

Each agent folder should include:

* **agent.py** – core agent logic
* **tools.py / tasks.py** – helper functions and integrations
* **README.md** – explanation of the agent's purpose and usage

---

## Design Principles

Agents in this repository follow a few key principles:

**Modularity**
Each agent should be independent and easy to expand.

**Tool usage**
Agents should be able to interact with external systems such as APIs, files, and databases.

**Iterative improvement**
Agents will evolve over time as new capabilities are tested.

**Practical usefulness**
Focus is on agents that solve real tasks rather than purely theoretical experiments.
